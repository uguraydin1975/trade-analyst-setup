Bu kurulum boyunca "terminale şunu yazın" diyeceğim. Terminal, bilgisayara
yazılı komutlar verdiğimiz siyah veya beyaz ekranlı bir penceredir.

Windows'ta terminal açmak:
  Yöntem 1 (En kolay):
    Windows tuşuna bas → arama kutusuna "cmd" yaz → "Komut İstemi"ne tıkla
  Yöntem 2:
    Windows + R tuşlarına aynı anda bas → "cmd" yaz → Enter'a bas
  Yöntem 3 (PowerShell):
    Windows tuşuna bas → "PowerShell" yaz → tıkla
    (PowerShell veya Komut İstemi — ikisi de çalışır)

Mac'te terminal açmak:
  Yöntem 1:
    Command + Boşluk → "Terminal" yaz → Enter
  Yöntem 2:
    Uygulamalar → Araçlar → Terminal

Terminal açıldıktan sonra:
  - Bir komut yazın, Enter'a basın, sonucu bekleyin
  - Hata mesajı çıkarsa tüm çıktıyı kopyalayıp bana yapıştırın
  - Terminali kapatmayın, kurulum boyunca açık tutun

Trade Analyst - Veri Çekme & İndikatör Hesaplama
600+ enstrüman tarar, 17 puanlık skor sistemi uygular.
Kullanım: python fetch_data.py --session morning

import pandas as pd
import pandas_ta as ta
from datetime import datetime, timezone
from typing import Optional

if sys.stdout.encoding != 'utf-8':
    sys.stdout.reconfigure(encoding='utf-8', errors='replace')
if sys.stderr.encoding != 'utf-8':
    sys.stderr.reconfigure(encoding='utf-8', errors='replace')

OUTPUT_DIR = os.path.join(os.path.expanduser("~"), "trade-analyst-output")
os.makedirs(OUTPUT_DIR, exist_ok=True)

CONFIG_PATH = os.path.join(os.path.dirname(__file__), "../config/instruments.json")
with open(CONFIG_PATH, encoding="utf-8") as f:
    INSTRUMENTS = json.load(f)

STABLECOINS = {
    "USDCUSDT","USD1USDT","FDUSDUSDT","UUSDT","XUSDUSDT","RLUSDUSDT",
    "PAXGUSDT","XAUTUSDT","EURUSDT","TUSDUSDT","BUSDUSDT","DAIUSDT"
}

# ── Sembol listeleri ──────────────────────────────────────────────────────────

def get_top50_crypto():
    try:
        r = requests.get("https://api.binance.com/api/v3/ticker/24hr", timeout=15)
        tickers = [t for t in r.json()
                   if t["symbol"].endswith("USDT")
                   and t["symbol"] not in STABLECOINS
                   and not any(x in t["symbol"] for x in ["UP","DOWN","BULL","BEAR"])]
        tickers.sort(key=lambda x: float(x["quoteVolume"]), reverse=True)
        syms = [t["symbol"] for t in tickers[:50]]
        print(f"  Kripto: {len(syms)} sembol çekildi")
        return syms
    except Exception as e:
        print(f"  [UYARI] Kripto listesi alınamadı: {e}")
        return []

def get_sp500_symbols():
    import urllib.request, csv, io
    for url in [
        "https://raw.githubusercontent.com/datasets/s-and-p-500-companies/main/data/constituents.csv",
        "https://datahub.io/core/s-and-p-500-companies/r/constituents.csv",
    ]:
        try:
            req = urllib.request.Request(url, headers={"User-Agent": "TradeBot/1.0"})
            resp = urllib.request.urlopen(req, timeout=20)
            reader = csv.DictReader(io.StringIO(resp.read().decode("utf-8")))
            syms = [row.get("Symbol","").strip() for row in reader if row.get("Symbol")]
            syms = [s for s in syms if 1 < len(s) <= 5]
            if len(syms) > 50:
                print(f"  S&P 500: {len(syms)} sembol çekildi")
                return syms
        except Exception as e:

def fetch_binance(symbol, interval="4h", limit=250):
    try:
        r = requests.get("https://api.binance.com/api/v3/klines",
                         params={"symbol": symbol, "interval": interval, "limit": limit},
                         timeout=10)
        r.raise_for_status()
        df = pd.DataFrame(r.json(), columns=[
            "ts","open","high","low","close","volume","ct","qv","t","tb","tq","i"
        ])[["ts","open","high","low","close","volume"]]
        df.columns = ["timestamp","open","high","low","close","volume"]
        df["timestamp"] = pd.to_datetime(df["timestamp"], unit="ms")
        for c in ["open","high","low","close","volume"]:
            df[c] = pd.to_numeric(df[c])
        return df.reset_index(drop=True)
    except Exception as e:
        return None

def fetch_yahoo(symbol, interval="1d", period_days=400):
    import urllib.request, urllib.parse, json as _j, time as _t
    end, start = int(_t.time()), int(_t.time()) - period_days * 86400
    url = (f"https://query1.finance.yahoo.com/v8/finance/chart/{urllib.parse.quote(symbol)}"
           f"?interval={interval}&period1={start}&period2={end}")
    try:
        req = urllib.request.Request(url, headers={"User-Agent": "Mozilla/5.0"})
        data = _j.loads(urllib.request.urlopen(req, timeout=15).read())
        r = data["chart"]["result"][0]
        q = r["indicators"]["quote"][0]
        df = pd.DataFrame({
            "timestamp": pd.to_datetime(r["timestamp"], unit="s"),
            "open": q["open"], "high": q["high"],
            "low": q["low"], "close": q["close"], "volume": q["volume"]
        }).dropna().reset_index(drop=True)
        return df
    except Exception as e:
        return None

def calculate_indicators(df):
    if df is None or len(df) < 30:
        return None
    close, high, low, volume, open_ = df["close"], df["high"], df["low"], df["volume"], df["open"]
    r = {}

    # RSI (analiz için — skorlamada yok)
    rsi = ta.rsi(close, length=14)
    r["rsi"] = round(float(rsi.iloc[-1]), 2) if rsi is not None else None

    # MACD ve kesişim tespiti
    macd_df = ta.macd(close, fast=12, slow=26, signal=9)
    if macd_df is not None:
        mc = [c for c in macd_df.columns if c.startswith("MACD_")][0]
        sc = [c for c in macd_df.columns if c.startswith("MACDs_")][0]
        hc = [c for c in macd_df.columns if c.startswith("MACDh_")][0]
        r["macd"], r["macd_signal"], r["macd_hist"] = (
            round(float(macd_df[mc].iloc[-1]),4),
            round(float(macd_df[sc].iloc[-1]),4),
            round(float(macd_df[hc].iloc[-1]),4))
        pm, ps = float(macd_df[mc].iloc[-2]), float(macd_df[sc].iloc[-2])
        cm, cs = r["macd"], r["macd_signal"]
        r["macd_cross"] = ("bullish_cross" if pm<ps and cm>cs
                      else "bearish_cross" if pm>ps and cm<cs
                      else "bullish" if cm>cs else "bearish")

    # EMA'lar ve trend
    for p in [20, 50, 200]:
        e = ta.ema(close, length=p)
        if e is not None: r[f"ema{p}"] = round(float(e.iloc[-1]),4)
    if all(f"ema{p}" in r for p in [20,50,200]):
        e20,e50,e200,px = r["ema20"],r["ema50"],r["ema200"],float(close.iloc[-1])
        r["ema_trend"] = ("guclu_yukselis" if e20>e50>e200 and px>e20
                     else "zayif_yukselis" if e20>e50
                     else "guclu_dusus" if e20<e50<e200 and px<e20
                     else "zayif_dusus")

    # ADX
    try:
        adx_df = ta.adx(high, low, close, length=14)
        if adx_df is not None:
            ac = [c for c in adx_df.columns if c.startswith("ADX_")][0]
            dp = [c for c in adx_df.columns if c.startswith("DMP_")][0]
            dn = [c for c in adx_df.columns if c.startswith("DMN_")][0]
            av = round(float(adx_df[ac].iloc[-1]),2)
            dv,nv = float(adx_df[dp].iloc[-1]),float(adx_df[dn].iloc[-1])
            r["adx"] = av
            r["adx_trend"] = ("guclu_yukselis" if av>25 and dv>nv
                          else "guclu_dusus" if av>25 and nv>dv else "yatay")
    except: pass

    # Bollinger Bands + Squeeze
    bb = ta.bbands(close, length=20, std=2)
    if bb is not None:
        uc=[c for c in bb.columns if c.startswith("BBU")][0]
        mc=[c for c in bb.columns if c.startswith("BBM")][0]
        lc=[c for c in bb.columns if c.startswith("BBL")][0]
        r["bb_upper"],r["bb_mid"],r["bb_lower"] = (
            round(float(bb[uc].iloc[-1]),4),
            round(float(bb[mc].iloc[-1]),4),
            round(float(bb[lc].iloc[-1]),4))
        r["bb_width"] = round((r["bb_upper"]-r["bb_lower"])/r["bb_mid"]*100,2) if r["bb_mid"] else 0
        if len(close)>=50:
            ws=[(float(bb[uc].iloc[i])-float(bb[lc].iloc[i]))/float(bb[mc].iloc[i])*100
                for i in range(-50,0) if float(bb[mc].iloc[i])!=0]
            if ws: r["bb_squeeze"] = r["bb_width"] <= sorted(ws)[int(len(ws)*0.2)]

    # Hacim ve yön
    vm = volume.rolling(20).mean().iloc[-1]
    if vm and vm>0:
        r["vol_ratio"] = round(float(volume.iloc[-1]/vm),2)
        r["vol_directional"] = "up" if float(close.iloc[-1])>=float(open_.iloc[-1]) else "down"

    # OBV divergence
    obv = ta.obv(close, volume)
    if obv is not None and len(obv)>=10:
        pt = float(close.iloc[-1])-float(close.iloc[-10])
        ot = float(obv.iloc[-1])-float(obv.iloc[-10])
        r["obv_divergence"] = ("bearish" if pt>0 and ot<0
                          else "bullish" if pt<0 and ot>0 else "none")

    # Mum formasyonları
    candles = {}
    try:
        for name in ["hammer","engulfing","doji","shootingstar","morningstar","eveningstar"]:
            p = ta.cdl_pattern(open_, high, low, close, name=name)
            if p is not None and p.iloc[-1]!=0:
                candles[name] = int(p.iloc[-1])
    except: pass
    r["candle_patterns"] = candles

    # ATR
    atr = ta.atr(high, low, close, length=14)
    if atr is not None: r["atr"] = round(float(atr.iloc[-1]),4)

    # Fibonacci
    fh,fl = float(high.tail(50).max()),float(low.tail(50).min())
    if fh>fl:
        r.update({"fib_high":round(fh,4),"fib_low":round(fl,4),
                  "fib_618":round(fh-(fh-fl)*0.618,4),
                  "fib_500":round(fh-(fh-fl)*0.500,4),
                  "fib_382":round(fh-(fh-fl)*0.382,4)})

    # Pivot
    ph,pl,pc = float(high.iloc[-2]),float(low.iloc[-2]),float(close.iloc[-2])
    pv = (ph+pl+pc)/3
    r.update({"pivot":round(pv,4),"resist1":round(2*pv-pl,4),
              "support1":round(2*pv-ph,4),"support2":round(pv-(ph-pl),4)})

    r["price"] = round(float(close.iloc[-1]),6)
    r["change_pct"] = round((r["price"]-float(close.iloc[-2]))/float(close.iloc[-2])*100,2)
    return r

def score_signal(ind, price, ind_weekly=None, ind_daily=None):
    ls, ss = 0, 0
    lr, sr = [], []

    # TREND (max 4)
    et = ind.get("ema_trend","")
    if "yukselis" in et:
        w=2 if "guclu" in et else 1; ls+=w; lr.append(f"EMA {et} (+{w})")
    elif "dusus" in et:
        w=2 if "guclu" in et else 1; ss+=w; sr.append(f"EMA {et} (+{w})")
    at = ind.get("adx_trend","yatay"); av = ind.get("adx",0) or 0
    if at=="guclu_yukselis": ls+=2; lr.append(f"ADX güçlü yükseliş {av:.0f} (+2)")
    elif at=="guclu_dusus":  ss+=2; sr.append(f"ADX güçlü düşüş {av:.0f} (+2)")

    # SİNYAL (max 4)
    mc = ind.get("macd_cross","")
    if mc=="bullish_cross":  ls+=2; lr.append("MACD tam kesişim yukarı (+2)")
    elif mc=="bullish":      ls+=1; lr.append("MACD bullish (+1)")
    elif mc=="bearish_cross":ss+=2; sr.append("MACD tam kesişim aşağı (+2)")
    elif mc=="bearish":      ss+=1; sr.append("MACD bearish (+1)")
    bc=[n for n,v in ind.get("candle_patterns",{}).items() if v>0]
    brs=[n for n,v in ind.get("candle_patterns",{}).items() if v<0]
    if bc:  ls+=2; lr.append(f"Bullish mum: {','.join(bc)} (+2)")
    if brs: ss+=2; sr.append(f"Bearish mum: {','.join(brs)} (+2)")

    # FİYAT (max 3)
    bbl,bbu = ind.get("bb_lower"),ind.get("bb_upper")
    if bbl and price<=bbl*1.005: ls+=1; lr.append("BB alt bandı (+1)")
    if bbu and price>=bbu*0.995: ss+=1; sr.append("BB üst bandı (+1)")
    if ind.get("bb_squeeze"):
        ls+=2; ss+=2
        lr.append("BB Squeeze (+2)"); sr.append("BB Squeeze (+2)")

    # HACİM (max 3)
    vr,vd = ind.get("vol_ratio",1) or 1, ind.get("vol_directional","")
    if vr>=1.5:
        pts=2 if vr>=2.5 else 1
        if vd=="up":   ls+=pts; lr.append(f"Yükseliş hacmi {vr:.1f}x (+{pts})")
        elif vd=="down":ss+=pts; sr.append(f"Düşüş hacmi {vr:.1f}x (+{pts})")
    od = ind.get("obv_divergence","none")
    if od=="bullish": ls+=1; lr.append("OBV bullish div (+1)")
    elif od=="bearish":ss+=1; sr.append("OBV bearish div (+1)")

    # MULTI-TIMEFRAME (max +2, min -2)
    ml, ms = 0, 0
    for lbl, ti in [("Haftalık",ind_weekly),("Günlük",ind_daily)]:
        if not ti: continue
        tl = "yukselis" in ti.get("ema_trend","") or "bullish" in ti.get("macd_cross","") or ti.get("adx_trend")=="guclu_yukselis"
        td = "dusus" in ti.get("ema_trend","") or "bearish" in ti.get("macd_cross","") or ti.get("adx_trend")=="guclu_dusus"
        if tl: ml+=1; lr.append(f"{lbl} LONG uyumu (+1)")
        if td: ms+=1; sr.append(f"{lbl} SHORT uyumu (+1)")
    ls+=ml; ss+=ms
    dom = "long" if ls>=ss else "short"
    if dom=="long"  and ms>0 and ml==0: ls=max(0,ls-2); lr.append("TF çelişiyor (-2)")
    if dom=="short" and ml>0 and ms==0: ss=max(0,ss-2); sr.append("TF çelişiyor (-2)")

    direction = "LONG" if ls>=ss else "SHORT"
    score = max(ls,ss)
    return {"direction":direction,"score":score,"long_score":ls,"short_score":ss,
            "reasons":(lr if direction=="LONG" else sr)}

def fetch_current_price(symbol, market=None):

def save_signals_from_analysis(signals, candidates, session):
    db = load_db(); now = datetime.now(timezone.utc).isoformat(); added = 0
    for sig in signals:

        })

def update_positions():
    db = load_db()
    open_pos = [s for s in db["signals"] if s["status"]=="open"]
    if not open_pos: print("Açık pozisyon yok."); return
    print(f"{len(open_pos)} açık pozisyon kontrol ediliyor...")
    today = datetime.now(timezone.utc).date()
    for pos in open_pos:
        price = fetch_current_price(pos["symbol"], pos.get("market"))
        if not price: continue
        pos["current_price"] = price
        e,sl,tp,d = pos["entry_price"],pos["stop_loss"],pos["target"],pos["direction"]
        pct = ((price-e)/e*100) if d=="LONG" else ((e-price)/e*100)
        pos["result_pct"] = round(pct,2)
        hit_t = (price>=tp if d=="LONG" else price<=tp)
        hit_s = (price<=sl if d=="LONG" else price>=sl)
        od = datetime.fromisoformat(pos["open_time"]).date()
        expired = (today-od).days > 10
        if hit_t:   pos.update({"status":"hit_target","close_date":today.isoformat(),"close_reason":"Hedef fiyata ulaşıldı"}); print(f"  ✅ {pos['symbol']} HEDEF → %{pct:+.2f}")
        elif hit_s: pos.update({"status":"hit_stop","close_date":today.isoformat(),"close_reason":"Stop-loss tetiklendi"}); print(f"  ❌ {pos['symbol']} STOP → %{pct:+.2f}")
        elif expired: pos.update({"status":"expired","close_date":today.isoformat(),"close_reason":"10 gün süre doldu"}); print(f"  ⏱ {pos['symbol']} SÜRE DOLDU → %{pct:+.2f}")
        else:
            days=(today-datetime.fromisoformat(pos["open_time"]).date()).days
            print(f"  📊 {pos['symbol']:12} {d:5} {e}→{price:.4f} %{pct:+.2f} ({days}.gün)")
        time.sleep(0.3)
    save_db(db); print(f"Veritabanı güncellendi.")

def get_news_summary(session):
    date_str = datetime.now(timezone.utc).strftime("%d %B %Y")

def analyze_batch(instruments, session):
    lbl = "Sabah" if session=="morning" else "Akşam"
    msg = client.messages.create(
        model="claude-sonnet-4-6", max_tokens=4000,
        system=SYSTEM_PROMPT,
        messages=[{"role":"user","content":
            f"{lbl} taraması — {len(instruments)} enstrüman:\\n\\n{json.dumps(instruments,ensure_ascii=False,indent=2)}\\n\\nEn güçlü sinyaller önce."}])
    raw = msg.content[0].text
    html_part, signals = raw, []
    if "```signals" in raw:
        parts = raw.split("```signals")
        html_part = parts[0].strip()
        try:
            js = parts[1].split("```")[0].strip()
            if not js.rstrip().endswith("]"):
                lb = js.rfind("}"); js = js[:lb+1]+"\\n]" if lb>0 else js
            signals = json.loads(js)
        except Exception as e:
            print(f"  [UYARI] JSON parse: {e}")
    return html_part, signals

def analyze_all(scan_results, session):
    all_c = scan_results.get("strong",[]) + scan_results.get("moderate",[])
    candidates = sorted(
        [c for c in all_c if c.get("score",0)>=MIN_SCORE_FOR_CLAUDE],
        key=lambda x: x.get("score",0), reverse=True
    )[:MAX_INSTRUMENTS]
    if not candidates:
        return {"analyses":[],"summary":"Yeterli sinyal bulunamadı.","news":"","signals":[],"candidates":[]}

    slim = [{
        "symbol":c["symbol"],"market":c["market"],"price":c["price"],
        "change_pct":c.get("change_pct"),"direction":c["direction"],"score":c["score"],
        "rsi":c.get("rsi"),"macd_cross":c.get("macd_cross"),
        "ema_trend":c.get("ema_trend"),"adx":c.get("adx"),"adx_trend":c.get("adx_trend"),
        "bb_squeeze":c.get("bb_squeeze"),"vol_ratio":c.get("vol_ratio"),
        "obv_divergence":c.get("obv_divergence"),
        "candle_patterns":c.get("candle_patterns",{}),
        "fib_618":c.get("fib_618"),"support1":c.get("support1"),"resist1":c.get("resist1"),
        "atr":c.get("atr"),"reasons":c.get("reasons",[]),
        "tf_weekly":c.get("tf_weekly"),"tf_daily":c.get("tf_daily")
    } for c in candidates]

    batches = [slim[i:i+INSTRUMENTS_PER_BATCH] for i in range(0,len(slim),INSTRUMENTS_PER_BATCH)]
    all_analyses, all_signals = [], []
    print(f"\\n  Claude analizi: {len(candidates)} enstrüman, {len(batches)} batch...")
    news = get_news_summary(session)

    for i, batch in enumerate(batches,1):
        if i>1: print("  Rate limit: 65sn bekleniyor..."); time.sleep(65)
        print(f"  Batch {i}/{len(batches)} ({len(batch)} enstrüman)...")
        try:
            h, s = analyze_batch(batch, session)
            all_analyses.append(h); all_signals.extend(s)
        except Exception as e:
            if "rate_limit" in str(e):
                print("  Rate limit — 70sn bekleyip tekrar..."); time.sleep(70)
                try: h,s = analyze_batch(batch,session); all_analyses.append(h); all_signals.extend(s)
                except Exception as e2: all_analyses.append(f"<!-- Batch {i} hata: {e2} -->")
            else: all_analyses.append(f"<!-- Batch {i} hata: {e} -->")

    try:
        smsg = client.messages.create(
            model="claude-sonnet-4-6", max_tokens=400,
            messages=[{"role":"user","content":
                f"Bu {len(candidates)} enstrüman analizini 2-3 cümleyle özetle (Türkçe): {[c['symbol'] for c in candidates]}"}])
        summary = smsg.content[0].text
    except: summary = f"{len(candidates)} enstrüman analiz edildi."

    return {"analyses":all_analyses,"summary":summary,"news":news,"signals":all_signals,"candidates":candidates}

ADIM 8 — TEST
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Sırayla çalıştır. Her komuttan önce terminalde trade-analyst klasöründe olduğunuzdan emin ol.

Windows'ta klasöre geçmek için:
  cd C:\Users\%USERNAME%\trade-analyst

Mac/Linux'ta:
  cd ~/trade-analyst

Sonra her komutu sırayla çalıştır, her birinde çıktıyı kullanıcıdan sor:

1. Tek sembol testi:
   python scripts/fetch_data.py --symbol BTCUSDT

   Beklenen çıktı: JSON formatında indikatörler ve skor.
   Hata alırsa: eksik kütüphane veya internet bağlantısı kontrol et.

2. Küçük tarama (sadece kripto):
   python scripts/fetch_data.py --session morning
   (BIST ve ABD listeleri boşsa sadece kripto tarar)

3. Tam analiz:
   python scripts/analyze_and_report.py --session morning

   Beklenen: HTML rapor oluştu + e-posta gönderildi.

Windows için adımları ver:

1. Başlat → "Görev Zamanlayıcı" → Temel Görev Oluştur
2. Ad: "Trade Sabah Tarama"
3. Tetikleyici: Günlük → Hafta içi (Pzt-Cum) → SORU C'deki sabah saati
4. Eylem: Program başlat → C:\\Users\\KULLANICI\\trade-analyst\\morning_scan.bat
5. İleri → "Kullanıcı oturum açmasa bile çalıştır" seç
6. Aynısını evening_scan.bat için tekrarla

Mac/Linux için:
crontab -e komutunu çalıştır, şunu ekle (09:00 sabah, 19:00 akşam için):
0 9 * * 1-5 cd ~/trade-analyst && python scripts/fetch_data.py --session morning >> logs/morning.log 2>&1 && python scripts/analyze_and_report.py --session morning >> logs/morning.log 2>&1
0 19 * * 1-5 cd ~/trade-analyst && python scripts/fetch_data.py --session evening >> logs/evening.log 2>&1 && python scripts/analyze_and_report.py --session evening >> logs/evening.log 2>&1

Kullanıcı hata alırsa şu çözümleri uygula:

UnicodeEncodeError:
  Windows'ta: set PYTHONUTF8=1 && set PYTHONIOENCODING=utf-8 komutunu batch dosyasına ekle

ModuleNotFoundError: anthropic/pandas/pandas_ta:
  pip install anthropic pandas pandas-ta requests

TA-Lib kurulum hatası:
  github.com/cgohlke/talib-build/releases → cp312 win_amd64 .whl indir → pip install dosya.whl

Yahoo Finance 404:
  Sembol .IS uzantısı kontrol et (BIST için AKBNK.IS formatı)
  Yahoo bazen geçici kapalı — 5 dakika bekle

Rate limit (429) Claude API:
  INSTRUMENTS_PER_BATCH = 3 yap, batch arası beklemeyi 90 saniyeye çıkar
  Uzun vadeli çözüm: console.anthropic.com'dan kredi yükle (ilk katman yeterli)

Gmail SMTP hatası:
  Normal Gmail şifresi değil, App Password kullanılmalı
  myaccount.google.com → Güvenlik → Uygulama şifreleri

n8n bağlantı hatası:
  n8n'in açık olduğundan emin ol: terminalde n8n start komutunu çalıştır
  Rapor her durumda kaydedilir (trade-analyst-output klasöründe)
  n8n açık değilse sistem otomatik olarak Gmail SMTP ile doğrudan gönderir
  n8n workflow'unda sorun varsa ilgili adımın ekran görüntüsünü gönder
